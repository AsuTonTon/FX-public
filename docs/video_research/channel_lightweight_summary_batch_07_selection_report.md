## 4. Methodology

The source pool is the high-priority caption-available unsummarized set.

Excluded:

- all 48 Batch 01-06 summarized video IDs
- the three caption-unavailable high-priority IDs
- rows selected only because of dramatic, profitable, or high-return title wording
- rows whose sampled caption evidence did not add useful doctrine coverage beyond Batch 01-06

Selection priority:

- TP, exit, and trade-management evidence
- stop or invalidation evidence
- entry confirmation
- trend and range context
- timeframe hierarchy
- false-break, damashi, or sweep handling
- horizontal-line construction versus line execution distinctions
- discretionary blockers
- indicator or session evidence only where caption-supported

Previously audited but deferred rows were reconsidered only where they added a useful boundary or doctrine control not already covered cleanly by Batch 01-06. Shorts were not excluded, but were limited to two selected rows because the regular-video corpus remains the primary evidence base.
