## 2. Main claim

The video argues that stop placement should be tied to the point where the trade premise fails. A horizontal line alone is not enough; the stop should follow the pattern, wave, or timeframe structure that justified entry.

## 3. Setup and context

The examples are framed around chart patterns and Elliott-wave-like wave counts. The setup context matters because the stop is selected from the same structure that created the entry idea.

## 4. Entry condition

The video does not define a complete independent entry trigger. It warns against entering without chart-pattern, wave, or similar grounds because a stop cannot be chosen coherently if the entry premise is vague.

## 5. Stop and invalidation

The stop belongs where the entry grounds fail. For pattern trades, the pattern's textbook stop area is prioritized. For wave examples, the invalidation anchor is described as the start of the relevant wave for a third-wave idea, or below the prior corrective low for a fifth-wave idea.

## 6. TP and exit logic

TP selection is not stated as a complete rule in the audited evidence. The stop discussion references the target's timeframe, but does not define a target-setting method.

## 8. Time-frame and session assumptions
