## 2. Main claim

The video defines waiting as a workflow: recognize the environment, build a scenario, set an alert at the pullback or return candidate, then enter only if a confirmation trigger appears.

## 3. Setup and context

Pullback candidates can be selected from a completed first wave, the last return high of a prior downtrend, smaller return points, support/resistance, and the amount of the first-wave break.

## 4. Entry condition

Entry is delayed until one of three triggers appears at the candidate area: chart pattern, horizontal-axis adjustment, or a candle that strongly breaks the previous candle. Touch-only entry is rejected.
