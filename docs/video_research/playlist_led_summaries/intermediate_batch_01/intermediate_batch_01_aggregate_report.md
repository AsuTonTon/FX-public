## 4. Intermediate doctrine synthesis

The accessible intermediate batch extends the completed beginner line in five main directions:

- Trade selection and suppression: weak discretionary entries can offset strong setups, so random trades should be filtered before increasing activity.
- Opportunity quality and sizing: larger risk belongs only to unusually clear setups, but the batch does not define a universal numeric sizing rule.
- Environment and wave context: trend, volatility, left-side wave visibility, active timeframe, horizontal buildup, and target reference levels shape whether an entry family is usable.
- Gold-specific context: anomaly timing, range breaks, bubble-style extension, fractal comparison, and short-timeframe adaptation are treated as gold-context tools rather than universal signals.
- Trader process and operating environment: broker/funding-route risk, fact-checking, test transfers, verbalized reasons, and goal-to-method alignment are treated as part of the trader's system.

Compared with the beginner synthesis, this batch is less about first principles and more about intermediate filters: when to trade, when not to trade, which timeframe should control the decision, and whether the market environment is worth engaging. The batch also strengthens the visual-discretion warning. Several concepts are useful as checklist gates, but wave boundaries, pattern location, fuel, range quality, and higher-timeframe priority still require manual chart judgment.

This synthesis is not final channel doctrine. It does not update Doctrine Map v1, does not create or finalize Doctrine Map v2, and does not treat any draft v2 material as final.
